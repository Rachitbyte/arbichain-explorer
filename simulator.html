Block Mining Simulator

Experience proof-of-work hashing and blockchain immutability hands-on.

🔐 Hash

A hash is a unique fingerprint for any data. Change one character and the entire hash changes completely.

🎲 Nonce

A nonce is a number miners increment until the hash output meets a target (e.g., starts with '00'). This is Proof of Work.

⛓️ Chain

Each block stores the previous block's hash. Changing Block 1 breaks Block 2's link — this is immutability.

💡 The Core Insight

When you changed Block 1's data above, Block 2 immediately became invalid. In a real blockchain with thousands of nodes, an attacker would need to re-mine not just Block 2, but every subsequent block — faster than the entire network combined. This is why blockchain data is effectively immutable.
